# User List Viewer

A mini React app that fetches and displays a list of users from the JSONPlaceholder API using a reusable list component.

## Features

- Fetches data using `fetch()` on component mount
- Displays user info in a styled list
- Reusable `<ListComponent />` accepts `items` and `renderItem`
- Graceful loading and error handling

## Getting Started

```bash
git clone https://github.com/yourname/user-list-app.git
cd user-list-app
npm install
npm start
```
